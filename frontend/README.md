### Local Frontend Development

#### Install tools and start Ganache

Install dependencies:

```sh
npm install -g ganache-cli truffle @graphprotocol/graph-cli
```

#### Connect this dApp to the subgraph

1. Start this app:
   ```sh
   npm install
   npm start
   ```
